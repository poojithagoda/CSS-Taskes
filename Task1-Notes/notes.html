CSS

CSS is a style language used to control the layout, colors, fonts, and overall appearance of HTML web pages.

CSS stands for CASCADING STYLE SHEET. CSS is used to add styles to html webpages. Cascading means flow of styles,specificity of elements and style sheet means adding css to the code is called cascading style sheet. The main goal of css is to make webpage or website more effective and presentable to the user. CSS is maintained by w3c (world wide web consortium). Hakon wium lie and bert bose are the inventors of the css. In 1994 oct 10 css 1st version came. Present css 3rd version we use.

Style refers to the visual presentation of html elements using CSS Properties and their corresponding values. Responsiveness is nothing but our webpage or website should change according to the screen size of the device. We use style in two ways that is tag and attributes based on the requirement we use this. If we want to add style to html we need particularly ,tag and attribute. Property values are nothing but property is like color,font-size,font-family. values like red,and lucida. Overall which type of styling we need to add.

We have 3 ways to insert css in html

  1. Inline css
  2. Internal css
  3. External css

1.Inline css:

Within the element we use inline css. we write this inline css within the body section. we use style attributes here.

2.Internal css:

Within the head section of html we write this internal css. using style tag we use this internal css.

3.External css:

need to create new file and save that file using .css extension. we dont need any attribute ,directly we write selector and property values. using link tag we can link this css file to html document.
